## 📌 Featured Projects

### 🏥 [Credit Risk Classification](https://github.com/Clivesay1/credit-risk-classification)
Built a logistic regression model to predict loan default risk with **99% accuracy** and 0.95 recall on high-risk loans. Demonstrates machine learning fundamentals applied to a real financial use case.

### 🏈 [NFL Special Teams Analytics](https://github.com/Clivesay1/Visualization-Project)
Analyzed 50 years of NFL kickoff data to measure the impact of rule changes on special teams performance. Built using Python, PostgreSQL, seaborn, and an interactive JavaScript/Flask dashboard.

### 👶 [Tennessee Childcare Cost Analysis](https://github.com/Clivesay1/Project-1)
Compared childcare affordability in Tennessee between 2018 and 2023 using U.S. Department of Labor data, Census API, and pandas. Identified key cost drivers for center-based vs. home-based care.

### 🏫 [PyCity Schools District Analysis](https://github.com/Clivesay1/Pandas-challenge)
Analyzed standardized test results and budget data across a school district to identify performance drivers. Found that school type (charter vs. district) is a stronger predictor of outcomes than per-student spending.

### 🌦️ [Hawaii Climate API](https://github.com/Clivesay1/sqlalchemy-challenge)
Built a Flask REST API serving climate data queried from a SQLite database using SQLAlchemy ORM. Includes precipitation analysis, station activity rankings, and temperature range endpoints.

### 👨‍💼 [Employee Database (SQL)](https://github.com/Clivesay1/SQL-Challenge)
Designed a 6-table PostgreSQL schema from scratch using ERD modeling, then wrote complex SQL queries to answer business questions about a historical employee dataset.
